Project JDK : Java 17

Instructions for importing app in eclipse and executing the project.

After downloading the project's zip file,

Open the eclipse, go to File -> import -> General -> Existing projects into workspace -> Select the archive file -> finish

After importing, for executing the project, go to Main.java and run the main method.

*********

For executing all test cases together or random tests seperately, first, the instructions in Readme file in blackbox\RandomTesting folder must be followed. Otherwise, after execution, random testing methods will wait user input from method.
